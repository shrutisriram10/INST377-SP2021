Learning Objectives

Attaching CSS to HTML to set up a page structure can be confusing the first few times you're asked to do it. In order to be able to complete Assignment 1, you'll need to be able to understand how CSS hooks to HTML. 

This lab should help you work out how CSS stylesheet rules bind to HTML element tags.

Collaborative or No?

This lab is intended to be completed on your own. While you can ask your classmates for help, please submit your own work for grading.

Lab Instructions

To successfully submit your labs, you will need to follow the steps laid out here:

  •  Required Lab Setup

Lab Setup

  1. Pull to update your version of the class repository
  2. Turn on your local server
    1. opening two terminal windows
    2. type npm install && npm start
    3. You should see a message saying something is running on Port 3000
    4. Check that this is happening by opening a browser and navigating to localhost:3000/lab_3/

Lab Submission

  • When you are done, you should be able to see your lab at https://yourusername.github.io/public/lab_3
  • Please submit your github.io link as your lab submission
